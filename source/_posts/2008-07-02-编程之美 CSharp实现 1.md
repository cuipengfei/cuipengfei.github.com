适用于1.8GHZ左右的CPU 50%占用率

```

class ManageCpu {
	staticvoid Main(string[] a) {
		int st = 0;
		while (true) {
			st = System.Environment.TickCount;
			
			while (System.Environment.TickCount - st < 90) {
			}
			
			System.Threading.Thread.Sleep(90);
		}
	}
}
```
