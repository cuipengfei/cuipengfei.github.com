语法101

*  每行代码都是分号结尾 

x=x+1  ；

*  注释以双斜线开始 

//this line is ignored

*  大多数空格都无所谓 

x    =     3  ；与x=3；一样

*  变量声明时都有类型和名字（第四章你会学习很多的类型） 

int weight;

//weight is an integer

*  类和方法定义时都有大括号 

public void go()

{

//amazing code here

}
