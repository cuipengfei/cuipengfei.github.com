🎮 演示控制

快捷键指南

基本控制

  • 空格 / → - 下一张
  • Shift + 空格 / ← - 上一张
  • F - 全屏模式
  • ESC - 退出概览

高级功能

  • S - 演讲者视图
  • O - 幻灯片概览
  • Alt + 点击 - 缩放
  • Ctrl + Shift + F - 搜索

演讲者视图

按 S 键打开演讲者视图，可以看到：

  • 当前幻灯片
  • 下一张幻灯片预览
  • 演讲者备注
  • 时间显示
  • 幻灯片计时
