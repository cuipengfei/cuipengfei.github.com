иҮӘе·ұеҠЁжүӢйҮҚж–°е®һзҺ°LINQ to Objects 7 - Countе’ҢLongCount

еҸ‘иЎЁдәҺ 2011-09-05 Disqusпјҡ
жң¬ж–Үеӯ—ж•°пјҡ 5.1k йҳ…иҜ»ж—¶й•ҝ ≈ 5 еҲҶй’ҹ

жң¬ж–Үзҝ»иҜ‘иҮӘ Jon Skeet зҡ„зі»еҲ—еҚҡж–ҮвҖқEdulinqвҖқгҖӮ

жң¬зҜҮеҺҹж–Үең°еқҖпјҡ

http://msmvps.com/blogs/jon_skeet/archive/2010/12/26/reimplementing-linq-to-objects-part-7-count-and-longcount.aspx

д»ҠеӨ©зҡ„ж–Үз« иҰҒд»Ӣз»ҚдёӨдёӘ LINQ ж“ҚдҪңз¬ҰпјҢеӣ дёәе®ғд»¬е®һеңЁжҳҜеӨӘзұ»дјјдәҶпјҢжүҖд»Ҙж”ҫеҲ°дёҖиө·жқҘи®ІгҖӮ Count е’Ң LongCount
зҡ„е®һзҺ°йқһеёёзӣёеғҸпјҢдёҚеҗҢзҡ„еҸӘжҳҜж–№жі•еҗҚпјҢиҝ”еӣһеҖјзұ»еһӢе’ҢеҮ дёӘеҸҳйҮҸгҖӮ

Count е’Ң LongCount жҳҜд»Җд№Ҳе‘ўпјҹ Count е’Ң LongCount
еҗ„иҮӘжңүдёӨдёӘйҮҚиҪҪпјҡдёҖдёӘйҮҚиҪҪжҺҘеҸ—и°“иҜҚпјҢеҸҰдёҖдёӘдёҚжҺҘеҸ—гҖӮдёӢйқўжҳҜиҝҷеӣӣдёӘж–№жі•зҡ„зӯҫеҗҚпјҡ

еҸҜд»ҘзңӢеҲ°пјҢ Count е’Ң LongCount зҡ„ж–№жі•зӯҫеҗҚзҡ„е·®еҲ«д»…еңЁдәҺиҝ”еӣһеҖјзұ»еһӢпјҢдёҖдёӘжҳҜ int пјҲ Int32 пјүпјҢдёҖдёӘжҳҜ longпјҲ Int64 пјүгҖӮ

дёҚжҺҘеҸ—и°“иҜҚзҡ„йҮҚиҪҪиҝ”еӣһиҫ“е…ҘеәҸеҲ—дёӯе…ғзҙ зҡ„дёӘж•°пјӣиҖҢжҺҘеҸ—и°“иҜҚзҡ„йҮҚиҪҪеҲҷиҝ”еӣһиғҪеӨҹйҖҡиҝҮи°“иҜҚйӘҢиҜҒзҡ„е…ғзҙ зҡ„дёӘж•°гҖӮ

иҝҷдәӣж–№жі•жңүдёҖдәӣжңүи¶Јзҡ„иЎҢдёәпјҡ

  • иҝҷеӣӣдёӘж–№жі•йғҪжҳҜжү©еұ• IEnumerable зҡ„ж–№жі•пјҢдҪ жңүеҸҜиғҪдјҡи®ӨдёәеҜ№дәҺдёҚжҺҘеҸ—и°“иҜҚзҡ„йӮЈдёӘйҮҚиҪҪжқҘиҜҙпјҢжү©еұ• IEnumerable
    дјҡжқҘеҫ—жӣҙеҘҪпјҢеӣ дёәжІЎжңүд»Җд№ҲдјҡйҷҗеҲ¶е…ғзҙ зҡ„зұ»еһӢгҖӮ

  • Count зҡ„дёҚжҺҘеҸ—и°“иҜҚзҡ„йӮЈдёӘйҮҚиҪҪеҜ№дәҺ ICollection е’Ң ICollection пјҲ .NET4
    дёӯзҡ„жҺҘеҸЈпјүеҒҡдәҶдјҳеҢ–пјҢеӣ дёәиҝҷдёӨдёӘжҺҘеҸЈйғҪе®ҡд№үжңү Count иҝҷдёӘеұһжҖ§пјҢиҝҷдёӘеұһжҖ§зҡ„е®һзҺ°еә”иҜҘжҜ”йҒҚеҺҶж•ҙдёӘйӣҶеҗҲиҰҒеҝ«гҖӮ LongCount
    еҲҷжІЎжңүеҒҡдјҳеҢ–пјҢзЁҚеҗҺзҡ„дёҖиҠӮдёӯжҲ‘е°Ҷдјҡи°ҲеҲ°иҝҷзӮ№гҖӮ

  • жҺҘеҸ—и°“иҜҚзҡ„йҮҚиҪҪдёӯжІЎжңүеҒҡд»»дҪ•дјҳеҢ–пјҢеӣ дёәдёҚиҝӯд»ЈжҜҸдёҖдёӘе…ғзҙ е°ұж— жі•зҹҘйҒ“еҲ°еә•жңүеӨҡе°‘дёӘе…ғзҙ еҸҜд»ҘйҖҡиҝҮи°“иҜҚзҡ„жЈҖйӘҢгҖӮ

  • иҝҷеӣӣдёӘж–№жі•йғҪжҳҜз«ӢеҚіжү§иЎҢзҡ„пјҢйғҪдёҚж¶үеҸҠе»¶иҝҹжү§иЎҢгҖӮпјҲд»”з»Ҷжғіжғіе°ұжҳҺзҷҪдәҶпјҢиҝҷдәӣж–№жі•д»…д»…иҝ”еӣһдёҖдёӘ int жҲ– long еҖјпјҢзЎ®е®һжІЎд»Җд№ҲеҸҜе»¶иҝҹжү§иЎҢзҡ„пјүгҖӮ

  • жүҖжңүзҡ„еҸӮж•°йғҪеҸӘеҒҡйқһ nullзҡ„ж ЎйӘҢгҖӮ

  • еҪ“иҫ“е…ҘйӣҶеҗҲзҡ„е…ғзҙ дёӘж•°и¶…еҮәдәҶ int жҲ– long зҡ„дёҠйҷҗеҖјж—¶пјҢеә”иҜҘжҠӣеҮә OverflowException гҖӮ

жҲ‘д»¬иҰҒжөӢиҜ•д»Җд№Ҳе‘ўпјҹ

жҲ‘д»¬йңҖиҰҒеҜ№ж–№жі•зҡ„дјҳеҢ–еҒҡжөӢиҜ•пјҢиҝҷд»¶дәӢеҒҡиө·жқҘжҜ”иҜҙиө·жқҘйҡҫпјҢеӣ дёәжҲ‘д»¬йңҖиҰҒжөӢиҜ•д»ҘдёӢеӣӣз§Қжғ…еҶөпјҡ

  • иҫ“е…ҘеәҸеҲ—еҗҢж—¶е®һзҺ°дәҶ ICollection е’Ң ICollection пјҲиҝҷдёӘз®ҖеҚ•пјҢзӣҙжҺҘз”Ё List пјү

  • иҫ“е…ҘеәҸеҲ—е®һзҺ°дәҶ ICollection дҪҶжІЎжңүе®һзҺ° ICollection пјҲиҝҳз®—з®ҖеҚ•пјҢеҸҜд»Ҙз”Ё HashSet пјү

  • иҫ“е…ҘеәҸеҲ—е®һзҺ°дәҶ ICollection дҪҶжІЎжңүе®һзҺ° ICollection пјҢжҲ‘иҝҳиҰҒиҰҒжұӮиҝҷдёӘзұ»еһӢе®һзҺ°дәҶ IEnumerable
    пјҲиҝҷж ·жүҚиғҪз”ЁеҲ°жҲ‘д»¬зҡ„жү©еұ•ж–№жі•пјүгҖӮ

  • иҫ“е…ҘеәҸеҲ—ж—ўдёҚе®һзҺ° ICollection д№ҹдёҚе®һзҺ° ICollection пјҲз®ҖеҚ•пјҢз”ЁжҲ‘д»¬е·Із»Ҹе®һзҺ°дәҶзҡ„ Range жқҘз”ҹжҲҗпјү

е…¶дёӯз¬¬дёүзӮ№жҜ”иҫғйә»зғҰгҖӮиҷҪз„¶жңүеҫҲеӨҡзұ»еһӢжҳҜе®һзҺ°дәҶ ICollection дҪҶжІЎжңүе®һзҺ° ICollection зҡ„пјҲжҜ”еҰӮ ArrayList
пјүпјҢдҪҶжҳҜе®ғд»¬йҖҡеёёд№ҹдёҚе®һзҺ° IEnumerable пјҢиҖҢжҲ‘д»¬зҡ„жү©еұ•ж–№жі•жҳҜй’ҲеҜ№дәҺ IEnumerable зҡ„гҖӮжүҖд»ҘжҲ‘еҸӘеҫ—иҮӘе·ұеҶҷдәҶдёҖдёӘеҸ«еҒҡ
SemiGenericCollection зҡ„зұ»гҖӮ

дёҠиҝ°зҡ„еӣӣз§Қиҫ“е…ҘеәҸеҲ—зҡ„зұ»еһӢйғҪжүҫеҲ°дәҶпјҢжҲ‘д»¬зҺ°еңЁйңҖиҰҒиҖғиҷ‘еҲ°еә•жҖҺд№ҲжөӢиҜ•дәҶгҖӮдҪ еҸҜиғҪдјҡиҜҙжҲ‘д»¬еҸҜд»ҘйҖҡиҝҮжЈҖжҹҘиҫ“е…ҘеәҸеҲ—жҳҜеҗҰиў«иҝӯд»ЈиҝҮжқҘжөӢиҜ•ж–№жі•жҳҜдёҚжҳҜзңҹзҡ„иў«дјҳеҢ–дәҶгҖӮдҪҶжҳҜиҰҒеҒҡиҝҷдёӘжөӢ
иҜ•е°ұйңҖиҰҒеҶҷдёҖдёӘжңү Count еҖјдҪҶжҳҜе…¶ GetEnumerator ж–№жі•дјҡжҠӣејӮеёёзҡ„йӣҶеҗҲзұ»еһӢгҖӮиҝҷдёӘжөӢиҜ•зЎ®е®һеҸҜиЎҢпјҢдҪҶжҳҜжҲ‘е№¶жІЎжңүеҒҡе®ғгҖӮ

еҜ№дәҺжҺҘеҸ—и°“иҜҚзҡ„йҮҚиҪҪжқҘиҜҙпјҢжҲ‘д»¬ж— йңҖиҖғиҷ‘йӮЈеҮ дёӘдёҚеҗҢзҡ„йӣҶеҗҲжҺҘеҸЈпјҢеӣ дёәжҲ‘д»¬еҸҚжӯЈйғҪдёҚдјҡдјҳеҢ–иҝҷдёӨдёӘж–№жі•гҖӮ

еҸӮж•°еҖјдёә nullзҡ„еҮ дёӘжөӢиҜ•йғҪжҜ”иҫғз®ҖеҚ•пјҢдҪҶжҳҜжңүеҸҰдёҖдёӘжөӢиҜ•жҜ”иҫғйҮҚиҰҒпјҡжәўеҮәгҖӮжҲ‘з»ҷ Count
еҲӣе»әдәҶдёҖдёӘжЈҖйӘҢжәўеҮәиЎҢдёәзҡ„еҚ•е…ғжөӢиҜ•гҖӮеҫҲдёҚе№ёпјҢжҲ‘д»¬зҺ°еңЁиҝҳдёҚиғҪеңЁ Edulinq зҡ„зҺҜеўғйҮҢиҝҗиЎҢе®ғпјҢеӣ дёәжҲ‘д»¬иҝҳжІЎжңүе®һзҺ° Concat
гҖӮдёҚиҝҮжҲ‘иҝҳжҳҜжҠҠе®ғеҶҷеңЁиҝҷйҮҢпјҡ

еҰӮжһң Count зҡ„е®һзҺ°еңЁеә”иҜҘжҠӣеҮәејӮеёёзҡ„ж—¶еҖҷжҠҠиҝ”еӣһеҖјжәўеҮәеҲ°дәҶ Int.MinValue зҡ„иҜқпјҢиҝҷдёӘжөӢиҜ•еҸҜд»ҘеҸ‘зҺ°еҲ°е®ғгҖӮ

дҪ еҸҜд»ҘзңӢеҲ°пјҢеҚідҪҝеңЁе®һзҺ°дәҶ Concat е№¶еҸҚжіЁйҮҠдәҶиҝҷж®өд»Јз Ғд№ӢеҗҺпјҢиҝҷдёӘжөӢиҜ•д№ҹжҳҜиў«еҝҪз•ҘжҺүдәҶзҡ„пјҢеӣ дёәе®ғйңҖиҰҒйҒҚеҺҶ 20
дәҝдёӘе…ғзҙ пјҢеҜ№дәҺеҮ дёӘз®ҖеҚ•еҝ«йҖҹзҡ„еҚ•е…ғжөӢиҜ•жқҘиҜҙпјҢиҝҷеҸҜдёҚеҰҷгҖӮе…¶е®һ 20 дәҝиҝҳдёҚз®—еӨӘеқҸпјҢеӣ дёә LongCount зҡ„жәўеҮәжөӢиҜ•йңҖиҰҒйҒҚеҺҶ 2 зҡ„ 63 ж¬Ў
ж–№дёӘе…ғзҙ е‘ўгҖӮиҰҒз”ҹжҲҗйӮЈд№Ҳй•ҝзҡ„еәҸеҲ—е№¶дёҚйҡҫпјҢйҡҫзҡ„жҳҜйҒҚеҺҶе®ғпјҢйӮЈиҰҒиҠұеҫҲй•ҝзҡ„ж—¶й—ҙгҖӮеҜ№дәҺжҺҘеҸ—и°“иҜҚзҡ„йҮҚиҪҪжқҘиҜҙпјҢжҲ‘д»¬д№ҹйңҖиҰҒеҒҡжәўеҮәжөӢиҜ•пјҢзӣҙеҲ°еҶҷиҝҷзҜҮж–Үз« д№ӢеүҚжҲ‘йғҪеҝҳи®°дәҶиҰҒеҶҷиҝҷдёӘжөӢиҜ•
пјҢиҖҢеҶҷжөӢиҜ•зҡ„ж—¶еҖҷиҝҳеҸ‘зҺ°дәҶдёҖдёӘж–№жі•е®һзҺ°дёӯзҡ„ bug :)

жқҘеҠЁжүӢе®һзҺ°еҗ§пјҒ

жҲ‘д»¬жқҘзңӢзңӢжҺҘеҸ—и°“иҜҚзҡ„йӮЈдёӘйҮҚиҪҪзҡ„е®һзҺ°еҗ§пјҢе®ғе…¶е®һжҢәз®ҖеҚ•зҡ„пјҡ

иҜ·жіЁж„ҸпјҢеңЁиҝҷйҮҢжҲ‘д»¬дёҚйңҖиҰҒиҝ”еӣһдёҖдёӘеәҸеҲ—пјҢжүҖд»Ҙе°ұжІЎжңүз”ЁеҲ°иҝӯд»ЈеҷЁд»Јз Ғеқ—пјҢеӣ иҖҢд№ҹе°ұж— йңҖжҠҠе®һзҺ°жӢҶеҲҶеҲ°дёӨдёӘж–№жі•дёӯеҺ»гҖӮ

еҸӮж•°ж ЎйӘҢд№ӢеҗҺзҡ„ж–№жі•дё»дҪ“йғЁеҲҶзӣёеҪ“з®ҖеҚ•пјҢеҸӘжңүдёҖзӮ№йңҖиҰҒжіЁж„Ҹпјҡж•ҙдёӘзҡ„иҝӯд»ЈиҝҮзЁӢйғҪеңЁвҖң checked вҖқд»Јз Ғеқ—дёӯгҖӮиҝҷж ·пјҢеҰӮжһң count
еҖјжәўеҮәзҡ„иҜқпјҢе°ұдјҡжҠӣеҮәејӮеёёпјҢиҖҢдёҚдјҡд»Ө count еҖјжҲҗдёәиҙҹж•°гҖӮд№ҹжңүе…¶д»–зҡ„ж–№ејҸеҸҜд»Ҙе®һзҺ°иҝҷзӮ№пјҡ

  • еҸҜд»ҘеҸӘжҠҠз»ҷ count еҠ дёҖзҡ„д»Јз Ғж”ҫеңЁ checked д»Јз Ғеқ—дёӯгҖӮ

  • еҸҜд»ҘеңЁжҜҸж¬Ўз»ҷ count еҠ дёҖд№ӢеүҚжЈҖжҹҘ count==int.MaxValue пјҢеҰӮжһңзЎ®е®һзӣёзӯүпјҢеҲҷжҠӣеҮәејӮеёё

  • еҸҜд»Ҙз»ҷж•ҙдёӘзЁӢеәҸйӣҶйғҪеә”з”Ё checked

жҲ‘и§үеҫ—жҠҠиҝҷж®өд»Јз ҒжҳҫејҸзҡ„ж”ҫеңЁ checked д»Јз Ғеқ—дёӯжҳҜеҫҲжңүзӣҠзҡ„пјҢеӣ дёәиҝҷж ·еҸҜд»ҘеҫҲжҳҺжҳҫзҡ„еҮёжҳҫеҮәеҜ№жәўеҮәзҡ„жЈҖжҹҘжҳҜж–№жі•жӯЈзЎ®жҖ§зҡ„йңҖжұӮиҝҷдёҖдәӢе®һгҖӮдҪ еҸҜиғҪжӣҙеҖҫеҗ‘дәҺеҸӘжҠҠ
count++ иҝҷдёҖеҸҘд»Јз Ғж”ҫеңЁ checked д»Јз Ғеқ—дёӯпјҢдёҚиҝҮжҲ‘дёӘдәәи§үеҫ—зҺ°еңЁзҡ„еҒҡжі•жӣҙе®№жҳ“еј•иө·иҜ»д»Јз Ғзҡ„дәәеҜ№ checked
зҡ„жіЁж„ҸпјҢеҪ“з„¶иҝҷеҸӘжҳҜжҲ‘зҡ„дё»и§ӮеҒҸеҘҪгҖӮиҝҳжңүпјҢжҳҫејҸзҡ„ checked д»Јз Ғеқ—жңүеҸҜиғҪдјҡеҝ«дёҖдәӣпјҢиҝҷдёҖзӮ№жҲ‘дёҚзЎ®е®ҡпјҢжҲ‘иҝҳжІЎжңүжөӢиҜ•иҝҮгҖӮ

йҷӨдәҶдёҺи°“иҜҚжңүе…ізҡ„йғЁеҲҶпјҢдёҠйқўзҡ„д»Јз Ғдјҡе…ЁйғЁеҮәзҺ°еңЁ Count зҡ„дјҳеҢ–иҝҮзҡ„е®һзҺ°дёӯпјҢжҲ‘д»¬е°ұдёҚеҶҚи®Іи§ЈдәҶпјҢзӣҙжҺҘеҶҷеҮәд»Јз Ғпјҡ

иҝҷдёӘе®һзҺ°йҮҢйқўе”ҜдёҖзҡ„вҖңж–°вҖқд»Јз Ғе°ұжҳҜе…ідәҺдјҳеҢ–зҡ„йӮЈж®өгҖӮдјҳеҢ–д»Јз Ғдёӯзҡ„дёӨж®өеҹәжң¬жҳҜдёҖж ·зҡ„пјҢе®ғд»¬жЈҖжҹҘдёҚеҗҢзҡ„йӣҶеҗҲжҺҘеҸЈзұ»еһӢпјҢе“ӘдёӘжЈҖжҹҘйҖҡиҝҮе°ұиҝ”еӣһе“ӘдёӘзҡ„ Count
еұһжҖ§гҖӮжҲ‘дёҚзҹҘйҒ“ .NET Framework зҡ„е®һзҺ°дёӯжҳҜе…ҲжЈҖжҹҘ ICollection иҝҳжҳҜе…ҲжЈҖжҹҘ ICollection
пјҢжҲ‘еҸҜд»ҘеҶҷдёҖдёӘеҗҢж—¶е®һзҺ°дәҶиҝҷдёӨдёӘжҺҘеҸЈпјҢдҪҶжҳҜеңЁдёӨдёӘ Count
еұһжҖ§дёӯиҝ”еӣһдёҚеҗҢзҡ„еҖјзҡ„зұ»еһӢжқҘжөӢиҜ•дёҖдёӢпјҢдҪҶжҳҜйӮЈд№ҲеҒҡжңүзӮ№еӨӘиҝҮдәҶгҖӮеҜ№дәҺиүҜеҘҪе®һзҺ°зҡ„йӣҶеҗҲжқҘиҜҙпјҢиҝҷзӮ№жҖ§иғҪе·®ејӮз®—дёҚдәҶд»Җд№ҲпјҢжҲ‘д»¬е…ҲжЈҖжҹҘвҖңжңҖжңүеҸҜиғҪвҖқзҡ„жҺҘеҸЈпјҢд№ҹе°ұжҳҜжіӣеһӢзҡ„
ICollection гҖӮ

дјҳеҢ–иҝҳжҳҜдёҚдјҳеҢ–пјҹ LongCount зҡ„е®һзҺ°е’Ң Count зҡ„е®һзҺ°еҮ д№Һе®Ңе…ЁдёҖж ·пјҢеҸӘжҳҜ LongCount дёӯдҪҝз”Ё long иҖҢдёҚжҳҜ int гҖӮ

жҲ‘еҜ№ ICollection е’Ң ICollection еҒҡдәҶдјҳеҢ–пјҢдҪҶжҳҜжҲ‘дёҚи®Өдёә .NET жҳҜеҰӮжӯӨе®һзҺ°зҡ„гҖӮпјҲеҸӘиҰҒеҲӣе»әдёҖдёӘеҫҲеӨ§зҡ„ byte
ж•°з»„пјҢе№¶жҜ”иҫғдёҖдёӢ Count е’Ң LongCount еә”з”ЁеҲ°иҝҷдёӘж•°з»„дёҠзҡ„иҖ—ж—¶е·®и·қе°ұеҸҜд»ҘдәҶгҖӮпјү

е…ідәҺ Array.GetLongCount зҡ„дҪҝз”ЁеӯҳеңЁдёҖдәӣдәүи®®пјҢдҪҶжҳҜжҲ‘и§үеҫ—ж—ўз„¶зҺ°еңЁзҡ„ CLR
е№¶дёҚж”ҜжҢҒй•ҝеәҰи¶…иҝҮ Int32.MaxValue зҡ„ж•°з»„пјҢйӮЈд№ҲиҝҷдёӘй—®йўҳзҺ°еңЁе°ұжІЎд»Җд№ҲеҘҪдәүи®®зҡ„пјҢеҸӘжңүз•ҷеҫ…д»ҘеҗҺиҜҒжҳҺгҖӮйҷӨжӯӨд№ӢеӨ–пјҢжҲ‘дёҚзҹҘйҒ“ .NET
зҡ„е®һзҺ°дёәд»Җд№ҲжІЎжңүдјҳеҢ–гҖӮеҜ№дёҖдёӘе®һзҺ°дәҶ ICollection жҲ– ICollection жҺҘеҸЈзҡ„зұ»еһӢжқҘиҜҙпјҢеҰӮжһңиҜҘйӣҶеҗҲдёӯеҢ…еҗ«дәҶи¶…иҝҮ
Int32.MaxValue дёӘе…ғзҙ зҡ„иҜқпјҢе®ғзҡ„ Count еұһжҖ§еә”иҜҘиҝ”еӣһд»Җд№ҲеҖје‘ўпјҹиҝҷдёҖзӮ№е№¶дёҚжҳҺзЎ®гҖӮ

ж¬ўиҝҺжҸҗеҮәеҗ„з§Қе»әи®®гҖӮдёҚиҝҮжҲ‘иҝҳжҳҜиҰҒжҢҮеҮә LongCount ж–№жі•еҸҜиғҪдјҡжӣҙеӨҡзҡ„еә”з”ЁеңЁ Queryable дёӯиҖҢдёҚжҳҜ Enumerable
дёӯпјҢиҺ·еҸ–дёҖдёӘж•°жҚ®еә“иЎЁзҡ„й•ҝеәҰзҡ„еңәжҷҜдјҡжҜ”иҫғеӨҡи§ҒпјҢиҖҢиҺ·еҸ–дёҖдёӘеҶ…еӯҳдёӯзҡ„йӣҶеҗҲзҡ„й•ҝеәҰзҡ„еңәжҷҜеҲҷдёҚйӮЈд№Ҳеёёи§ҒгҖӮ

з»“и®ә

иҝҷжҳҜжҲ‘д»¬з¬¬дёҖж¬ЎжҺҘи§Ұиҝ”еӣһж•°еҖјиҖҢдёҚжҳҜиҝ”еӣһдёҖдёӘеәҸеҲ—зҡ„ LINQ ж“ҚдҪңз¬ҰпјҢеҫҲиҮӘз„¶пјҢиҝҷж ·зҡ„ж“ҚдҪңз¬Ұжӣҙе®№жҳ“зҗҶи§ЈгҖӮиҝҷдәӣж–№жі•еҫҲз®ҖеҚ•зҡ„жү§иЎҢпјҢеҒҡдёҖдәӣдјҳеҢ–пјҢз„¶еҗҺиҝ”еӣһз»“жһңеҖјгҖӮиҝҷдәӣ
ж–№жі•иҷҪз„¶з®ҖеҚ•пјҢдҪҶжҳҜиҝҳжҳҜжңүдёҖдәӣдёңиҘҝеҫҲеҖјеҫ—жҖқиҖғпјҢжҜ”еҰӮиҜҙдјҳеҢ–зҡ„й—®йўҳпјҢеҸҜжғңдјҳеҢ–зҡ„й—®йўҳе№¶жІЎжңүдёҖдёӘзЎ®е®ҡзҡ„зӯ”жЎҲгҖӮ

дёӢдёҖж¬ЎжҲ‘жғіжҲ‘дјҡеҺ»е®һзҺ° Concat пјҢдё»иҰҒжҳҜеӣ дёәе®һзҺ°дәҶ Concat е°ұеҸҜд»ҘжҠҠеҜ№ Count иҝӣиЎҢжәўеҮәжөӢиҜ•зҡ„д»Јз ҒеҸҚжіЁйҮҠдәҶгҖӮ Concat
